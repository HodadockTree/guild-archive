# 냥춘 길드 활동 아카이브

현재 버전: v1.5

테일즈런너 길드 `냥춘`의 활동 기록을 관리하고 보관하기 위한 모바일 중심 웹앱입니다.

단순 길드원 관리 도구가 아니라, 길드 활동을 기록하고 월별로 정산해 나중에 다시 볼 수 있도록 하는 **길드 활동 아카이브**를 목표로 합니다.

## 주요 기능

### 길드원 관리

- 활동중 길드원과 탈퇴 길드원을 분리해서 관리
- 탈퇴 길드원도 과거 활동 참여자로 기록 가능
- 활동 기록이 없는 길드원 삭제 지원
- 길드원별 활동 요약 통계 확인
- 길드원별 활동 이력 모달 확인
- 활동 이력이 많은 길드원도 메인 화면 레이아웃이 길어지지 않도록 개선

### 활동 기록

- 점령전 / 비공정 / 이벤트 활동 기록
- 기존 `기타` 활동은 사용자 화면에서 `이벤트`로 표시
- 기존 과거 기록과의 호환 유지
- 비공정 세부 종류 기록
  - 오션헤븐
  - 아우로라
- 점령전 세부 카테고리 기록
  - 용기
  - 신념
  - 평화
  - 신성
  - 지혜
  - 예언
  - 초심
  - 긍지
  - 역전
- 점령전 세부 카테고리 다중 선택 지원
- 참여자 검색 및 초성 검색 지원
- 활동 기록 최신순 / 오래된순 정렬
- 활동 기록 수정 / 삭제 지원
- 활동 기록 추가 / 수정 / 삭제 완료 메시지 표시
- 활동 기록 카드에 참여 인원 수 표시

### 월별 활동 정산

- 월 선택 가능한 활동 정산 대시보드
- 월별 전체 활동 수 표시
- 총 참여 횟수 표시
- 참여 길드원 수 표시
- 비공정 / 점령전 / 이벤트 통계 표시
- 오션헤븐 / 아우로라 비공정 세부 통계 표시
- 점령전 세부 카테고리 통계 표시
- 월간 참여 TOP N 표시
- 이번 달 활동 기록 요약 표시
- 최다 참여 활동 배지 표시
- 월별 활동 정산 공유 문구 복사 지원

### 공유 기능

- 선택한 월의 활동 정산 공유 문구 생성
- 공유 문구 복사 버튼 지원
- 공유 문구에는 다음 정보 포함
  - 전체 활동 수
  - 총 참여 횟수
  - 참여 길드원 수
  - 활동별 기록
  - 참여 TOP
  - 이번 달 이벤트 목록
- 월별 정산 설정 영역에서 공유 문구 복사 가능
- 캡처용 정산 카드에는 조작 버튼이 보이지 않도록 구성

### 입력 UX

- 비공정 입력 시 오션헤븐 / 아우로라 버튼형 입력 지원
- 점령전 입력 시 세부 카테고리 선택 중심으로 입력
- 점령전 빠른 제목 버튼 제거
- 제목 직접 입력과 기존 제목 데이터 호환 유지
- 선택한 참여자 수 표시
- 참여자 목록에서 현재 활동 종류 기준 참여 횟수 표시

### 데이터 관리

- 브라우저 LocalStorage 기반 데이터 저장
- 전체 데이터 JSON 백업 내보내기
- JSON 백업 파일 가져오기
- 백업 파일 구조 검증
- 복원 전 백업 파일 요약 표시
- 2단계 확인 후 덮어쓰기 복원
- 잘못된 파일 / 복원 취소 시 기존 데이터 유지

## 데이터 저장 방식

이 앱은 별도 서버 없이 브라우저의 LocalStorage에 데이터를 저장합니다.

따라서 브라우저 데이터 삭제, 기기 변경, 시크릿 모드 사용 등에 따라 데이터가 사라질 수 있습니다.

중요한 데이터는 주기적으로 JSON 백업을 내보내 보관하는 것을 권장합니다.

## 현재 활동 종류 기준

사용자 화면 기준 활동 종류는 다음과 같습니다.

- 점령전
- 비공정
- 이벤트

기존 버전에서 사용하던 `기타` 활동은 v1.5부터 사용자 화면에서 `이벤트`로 표시됩니다.

다만 기존 LocalStorage 데이터와 JSON 백업 호환을 위해 내부 데이터 구조는 유지됩니다.

## 향후 고려 사항

- 공유 문구 톤 개선
- 이벤트 기록 모아보기
- 길드 연혁 페이지
- 이벤트 중심 기록 보관 기능
- schemaVersion 기반 마이그레이션
- 내부 활동 타입 정식 정리
