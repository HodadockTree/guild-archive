# 냥춘 길드 활동 아카이브

테일즈런너 길드 "냥춘"의 활동 기록을 개인적으로 보관하기 위한 웹앱입니다.

길드원 관리 시스템을 목표로 하기보다는, 길드 활동이 언제 있었고 누가 참여했는지 남겨두는 가벼운 아카이브에 초점을 둡니다. 서버나 로그인 없이 브라우저의 LocalStorage에 데이터를 저장합니다.

## 주요 기능

- 길드원 등록, 정보 수정, 탈퇴 처리
- 탈퇴 길드원 기록 보존 및 일괄 복구
- 스프레드시트에서 길드원 일괄 가져오기
- 최근 일괄 가져오기 되돌리기
- 활동 기록 추가, 수정, 삭제
- 활동 종류 선택: 비공정, 점령전, 길드퀘, 이벤트, 기타 메모
- 활동 제목 빠른 입력
- 활동 참여자 체크
- 참여자 목록 빈도순 정렬
- 활동 기록 스크린샷 첨부
- 디스코드 이미지 Ctrl+V 첨부
- 전체 활동 기록 목록 보기
- 활동 종류별 필터
- 길드원별 활동 이력 보기
- 활동중/탈퇴 길드원 분리 보기
- 길드원 관리 영역 접기/펼치기
- LocalStorage 저장

## 기술 스택

- Next.js
- React
- TypeScript
- LocalStorage

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 확인합니다.

## 개발 원칙

- 개인 실사용 흐름을 우선합니다.
- 서버 없이 LocalStorage 기반으로 동작합니다.
- 탈퇴한 길드원도 과거 활동 기록에서 삭제하지 않습니다.
- 과거 기록을 복원할 때 기존 활동 이력이 깨지지 않도록 주의합니다.
- 실제로 필요한 기능만 작게 추가합니다.
- 기록 보존과 유지보수성을 과한 구조화보다 우선합니다.

## 현재 상태

v1.1 기준으로 과거 기록 복원과 실사용 입력 편의성 개선까지 반영되어 있습니다.

길드원 일괄 가져오기, 가져오기 되돌리기, 탈퇴 길드원 복구, 스크린샷 첨부, 디스코드 이미지 붙여넣기, 참여자 빈도순 정렬, 빠른 제목 입력을 사용할 수 있습니다.

데이터와 첨부 이미지는 현재 브라우저의 LocalStorage에만 저장되므로, 브라우저나 저장소를 초기화하면 기록이 사라질 수 있습니다.
