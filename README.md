# 냥춘 길드 활동 아카이브

테일즈런너 길드 "냥춘"의 활동 기록을 개인적으로 보관하기 위한 웹앱입니다.

길드원 관리 시스템을 목표로 하기보다는, 길드 활동이 언제 있었고 누가 참여했는지 남겨두는 가벼운 아카이브에 초점을 둡니다. 서버나 로그인 없이 브라우저의 LocalStorage에 데이터를 저장합니다.

## 주요 기능

- 길드원 등록, 정보 수정, 탈퇴 처리
- 활동중/탈퇴 길드원 분리 보기
- 스프레드시트 길드원 일괄 가져오기 및 최근 가져오기 되돌리기
- 탈퇴일 기반 탈퇴 상태 반영
- 길드원 메모 일괄 삭제
- 활동 기록 추가, 수정, 삭제
- 활동 종류 기록: 점령전, 비공정, 기타
- 비공정 세부 종류 기록: 오션헤븐, 아우로라
- 활동 참여자 검색 및 체크
- 선택한 활동 종류 기준 참여 횟수 표시
- 길드원별 활동 요약 통계와 최근 활동 확인
- 활동 기록 스크린샷 첨부
- 디스코드 이미지 Ctrl+V 첨부
- 전체 활동 기록 목록 보기와 활동 종류별 필터
- 길드원별 활동 이력 보기
- LocalStorage 저장

## 기술 스택

- Next.js
- React
- TypeScript
- LocalStorage

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 확인합니다.

## 개발 원칙

- 개인 실사용 흐름을 우선합니다.
- 서버 없이 LocalStorage 기반으로 동작합니다.
- 탈퇴한 길드원도 과거 활동 기록에서 삭제하지 않습니다.
- 과거 기록을 복원할 때 기존 활동 이력이 깨지지 않도록 주의합니다.
- 실제로 필요한 기능만 작게 추가합니다.
- 기록 보존과 유지보수성을 과한 구조화보다 우선합니다.

## 현재 상태

현재 버전은 v1.2입니다.

v1.2에서는 활동 기록 구조와 길드원별 활동 요약 통계를 정리하고, 스프레드시트 가져오기와 기존 데이터 정리 흐름을 안정화했습니다.

데이터와 첨부 이미지는 현재 브라우저의 LocalStorage에만 저장되므로, 브라우저나 저장소를 초기화하면 기록이 사라질 수 있습니다.
