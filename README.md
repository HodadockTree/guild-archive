# 냥춘 길드 활동 아카이브

테일즈런너 길드 "냥춘"의 활동 기록을 개인적으로 보관하기 위한 웹앱입니다.

길드원 관리 시스템을 목표로 하기보다는, 길드 활동이 언제 있었고 누가 참여했는지 남겨두는 가벼운 아카이브에 초점을 둡니다. 서버나 로그인 없이 브라우저의 LocalStorage에 데이터를 저장합니다.

## 주요 기능

- 길드원 등록, 정보 수정, 탈퇴 처리
- 활동 기록 없는 길드원 삭제
- 활동중/탈퇴 길드원 분리 보기
- 스프레드시트 길드원 일괄 가져오기 및 최근 가져오기 되돌리기
- 길드원 메모 일괄 삭제
- 활동 기록 추가, 수정, 삭제
- 활동 기록 최신순/오래된순 정렬
- 활동 종류 기록: 점령전, 비공정, 기타
- 비공정 버튼형 입력: 오션헤븐 비공정, 아우로라 비공정
- 활동 참여자 검색 및 초성 검색
- 선택 참여자 수와 활동별 참여 횟수 표시
- 길드원별 활동 요약 통계와 상세 통계 확인
- 월별 활동 정산 대시보드
- 월간 참여 TOP 5와 활동 기록 요약
- 전체 데이터 JSON 백업 내보내기
- 활동 기록 스크린샷 첨부
- 디스코드 이미지 Ctrl+V 첨부
- 데이터 관리 도구
- LocalStorage 저장

## 기술 스택

- Next.js
- React
- TypeScript
- LocalStorage

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 확인합니다.

## 개발 원칙

- 개인 실사용 흐름을 우선합니다.
- 서버 없이 LocalStorage 기반으로 동작합니다.
- 탈퇴한 길드원도 과거 활동 기록에서 삭제하지 않습니다.
- 삭제 기능은 오타나 중복 등록처럼 잘못 만든 데이터를 정리할 때만 사용합니다.
- 과거 기록을 복원할 때 기존 활동 이력이 깨지지 않도록 주의합니다.
- 실제로 필요한 기능만 작게 추가합니다.

## 현재 상태

현재 버전은 v1.3입니다.

v1.3에서는 월별 활동 정산 대시보드와 전체 데이터 JSON 백업 내보내기가 추가되었습니다. 공유용 텍스트 생성, 이미지 저장, JSON 복원은 아직 구현되어 있지 않습니다.

데이터와 첨부 이미지는 현재 브라우저의 LocalStorage에만 저장되므로, 브라우저나 저장소를 초기화하면 기록이 사라질 수 있습니다.
