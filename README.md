# 냥춘 길드 활동 아카이브

현재 버전: v1.3.1

## 주요 기능

### 길드원 관리

- 활동중 길드원과 탈퇴 길드원을 분리해서 관리
- 탈퇴 길드원도 과거 활동 참여자로 기록 가능
- 활동 기록이 없는 길드원 삭제 지원
- 길드원별 활동 요약 통계 확인

### 활동 기록

- 점령전 / 비공정 / 기타 활동 기록
- 비공정 세부 종류 기록
  - 오션헤븐
  - 아우로라
- 점령전 세부 카테고리 기록
  - 용기
  - 신념
  - 평화
  - 신성
  - 지혜
  - 예언
  - 초심
  - 긍지
  - 역전
- 점령전 세부 카테고리 다중 선택 지원
- 참여자 검색 및 초성 검색 지원
- 활동 기록 최신순 / 오래된순 정렬
- 활동 기록 수정 / 삭제 지원

### 월별 활동 정산

- 월 선택 가능한 활동 정산 대시보드
- 전체 활동 수 표시
- 참여 길드원 수 표시
- 총 참여 횟수 표시
- 비공정 / 점령전 / 기타 활동 통계 표시
- 오션헤븐 / 아우로라 비공정 세부 통계 표시
- 월간 참여 TOP N 표시
- 이번 달 활동 기록 요약 표시
- 최다 참여 활동 배지 표시
- 캡처해서 길드 톡방에 공유하기 좋은 대시보드 레이아웃

### 데이터 관리

- LocalStorage 기반 데이터 저장
- 전체 데이터 JSON 백업 내보내기
- 백업 JSON에 길드원, 활동 기록, 앱 버전, 스키마 버전, 백업 생성일 포함
