# CHANGELOG

## v1.2.1 - 활동 입력 및 관리 UX 보정

### 목표

- 실제 데이터 입력 중 발견한 불편함 개선
- 과거 활동 입력 속도 향상
- 활동 기록 확인과 수정 안정성 개선
- 오타/중복 길드원 데이터 정리 지원

### 구현

- 활동 기록 추가/수정/삭제 완료 메시지 추가
- 전체 활동 기록 최신순/오래된순 정렬 추가
- 활동 기록 카드에 참여 인원 수 표시
- 활동 기록 수정/삭제 버튼 가시성 개선
- 활동 기록 없는 길드원 삭제 기능 추가
- 활동 기록이 있는 길드원 삭제 방지
- 삭제 전 확인창과 삭제 완료 메시지 추가
- 활동 기록 작성 중 선택한 참여자 수 표시
- 비공정 입력을 오션헤븐/아우로라 버튼형으로 개선
- 비공정 버튼 클릭 시 `airshipType`과 제목을 함께 설정
- 참여자 초성 검색 추가
- 닉네임 일부 검색과 영문/숫자 검색 유지
- 검색 중에도 참여자 선택 상태 유지
- 탈퇴 길드원 복구 기능을 데이터 관리 도구 내부로 이동
- 길드원 카드 내부 활동 이력 UI 단순화

### 결과

- 과거 활동 입력 중 선택 인원과 기록된 참여 인원 확인이 쉬워졌습니다.
- 비공정 기록 입력 속도가 빨라졌습니다.
- 초성 검색으로 참여자 선택이 편해졌습니다.
- 활동 기록 추가/수정/삭제 결과를 더 명확히 확인할 수 있습니다.
- 오타로 잘못 등록한 활동 기록 없는 길드원을 안전하게 삭제할 수 있습니다.

### 다음 개선 후보

아래 항목은 아직 구현 완료가 아니라 다음 버전에서 검토할 후보입니다.

- 월별 활동 정산 / 공유용 요약
- 전체 데이터 백업/복원
- 길드원별 활동 이력 모달/패널화
- 활동 기록 검색/필터 고도화

## v1.2

### 목표

활동 기록 시스템을 정리하고, 기존 데이터 이관과 정리 과정을 더 안정적으로 만듭니다.

과거 기록을 입력하거나 복원할 때 실수를 줄이고, 길드원별 활동 현황을 화면 안에서 빠르게 확인할 수 있게 하는 것이 목표입니다.

### 구현 기능

- 스프레드시트 가져오기에서 탈퇴일 반영
- 탈퇴일이 있는 길드원을 탈퇴 상태로 처리
- 기존 닉네임과 중복될 경우 새로 만들지 않고 기존 길드원 업데이트
- 가져오기 결과 요약 개선: 추가, 업데이트, 탈퇴 처리, 실패
- 가져오기 실패 상세 표시
- 길드원 메모 일괄 삭제
  - 활동중, 탈퇴, 전체 범위 선택
  - 실행 전 확인
  - 메모 외 데이터 보존
- 신규 활동 종류를 점령전, 비공정, 기타 중심으로 정리
- 기존 길퀘, 이벤트 기록은 데이터 호환을 위해 유지하되 표시/통계에서는 기타로 취급
- 비공정 세부 종류 추가: 오션헤븐, 아우로라
- 길드원별 활동 통계 계산 로직 분리
- 길드원 카드에 활동 요약 통계 표시: 총 참여, 점령전, 비공정, 기타
- 카드 펼침 영역에 비공정 상세 통계 표시: 오션헤븐, 아우로라
- 활동 추가/수정 폼에서 선택한 활동 종류 기준 참여 횟수 표시
- 길드원 카드의 활동 이력 보기 버튼을 카드 내부 펼침 방식으로 변경
- 카드 펼침 영역에서 활동 요약과 최근 활동 5개 표시
- 활동 추가/수정 폼에 참여자 검색 추가
- 활동중/탈퇴 길드원 영역 분리
- 활동중 길드원 기본 펼침, 탈퇴 길드원 기본 접힘
- 검색 시 활동중/탈퇴 길드원 모두 검색
- 탈퇴 길드원이 포함된 기존 활동 수정 시 선택 상태 유지
- 스프레드시트 가져오기, 가져오기 되돌리기, 메모 일괄 삭제를 데이터 관리 도구 영역으로 정리
- 데이터 관리 도구 영역 기본 접힘 처리

### 결과

v1.2에서는 스프레드시트 자료를 가져올 때 기존 길드원과 더 안전하게 병합할 수 있습니다.

탈퇴일이 있는 길드원은 자동으로 탈퇴 상태가 되고, 중복 닉네임은 새 길드원을 만들지 않고 기존 길드원 정보 업데이트로 처리됩니다.

활동 기록은 점령전, 비공정, 기타 중심으로 단순화되었고, 비공정은 오션헤븐과 아우로라를 구분해서 기록할 수 있습니다.

길드원 카드에서 총 참여, 점령전, 비공정, 기타 참여 횟수와 최근 활동을 바로 확인할 수 있습니다.

### 다음 개선 후보

아래 항목은 아직 구현 완료가 아니라 다음 버전에서 검토할 후보입니다.

- LocalStorage 데이터 백업/복원
- 첨부 이미지 용량 관리 방식 개선
- 활동 기록 검색
- 활동 기록 정렬 옵션
- 읽기 전용 조회 화면 검토

## v1.1

### 목표

과거 기록 복원과 실사용 입력 편의성을 개선합니다.

기존에 흩어져 있던 길드원 기록을 더 쉽게 가져오고, 활동 기록 작성 중 반복 입력을 줄이며, 스크린샷 기반 기록도 함께 남길 수 있게 하는 것이 목표입니다.

### 구현 기능

- 길드원 정보 수정
- 활동 기록 스크린샷 첨부
- 디스코드 이미지 Ctrl+V 첨부
- 스프레드시트 길드원 일괄 가져오기
- 일괄 가져오기 되돌리기
- 탈퇴 길드원 일괄 복구
- 참여자 목록 빈도순 정렬
- 활동중/탈퇴 길드원 분리
- 길드원 관리 영역 접기/펼치기
- 활동 제목 빠른 입력 버튼

### 결과

v1.1에서는 과거 길드원 목록을 빠르게 복원하고, 잘못 가져온 최근 일괄 추가분을 되돌릴 수 있습니다.

탈퇴 상태로 남아 있던 길드원을 한 번에 활동중으로 복구할 수 있어, 과거 기록을 다시 이어서 쓰기 쉬워졌습니다.

활동 기록에는 스크린샷을 첨부할 수 있고, 디스코드에서 복사한 이미지를 Ctrl+V로 바로 붙여넣을 수 있습니다.

참여자 선택 목록은 기존 참여 빈도가 높은 길드원이 먼저 보이도록 정렬되어 반복 입력이 줄었습니다.

### 다음 개선 후보

아래 항목은 아직 구현 완료가 아니라 다음 버전에서 검토할 후보입니다.

- LocalStorage 데이터 내보내기/가져오기
- 활동 목록 검색
- 활동 기록 정렬 옵션
- 간단한 참여 횟수 요약
- 첨부 이미지 용량 관리 방식 개선

## v1.0

### 목표

냥춘 길드 활동을 개인적으로 기록하고 다시 확인할 수 있는 LocalStorage 기반 아카이브를 만듭니다.

길드원 관리 시스템보다는 활동 기록 보존에 집중합니다.

### 구현 기능

- 길드원 등록
- 길드원 탈퇴 처리
- 탈퇴 길드원 기록 보존
- 활동 기록 추가
- 활동 종류 선택
  - 비공정
  - 점령전
  - 길드퀘
  - 이벤트
  - 기타 메모
- 활동 참여자 체크
- 전체 활동 기록 목록 보기
- 활동 종류별 필터
- 활동 기록 수정
- 활동 기록 삭제
- 길드원별 활동 이력 보기
- LocalStorage 저장

### 결과

v1.0에서는 길드 활동을 날짜, 종류, 제목, 메모, 참여자 기준으로 남기고 조회할 수 있습니다.

탈퇴한 길드원도 삭제하지 않고 상태만 변경하므로, 과거 활동 기록의 참여자 이력이 유지됩니다.
