# CHANGELOG

## v1.5 - 월간 정산 공유 및 활동 이력 UX 개선

### 목표

월별 활동 정산 데이터를 길드 톡방에 쉽게 공유할 수 있도록 하고,  
길드원별 활동 이력 확인 시 메인 화면 레이아웃이 길어지는 문제를 개선한다.

또한 기존 `기타` 활동 표현을 길드 운영 기록에 더 자연스러운 `이벤트`로 정리한다.

### 구현

- 기존 `기타` 활동 표시명을 사용자 화면에서 `이벤트`로 정리
- 기존 기타 / 이벤트 호환 데이터는 유지하면서 화면 표시와 통계 라벨만 정리
- 월별 활동 정산 공유 문구 생성 기능 추가
- 월별 정산 설정 영역에 `공유 문구 복사` 버튼 추가
- 공유 문구에 다음 정보 포함
  - 전체 활동 수
  - 총 참여 횟수
  - 참여 길드원 수
  - 활동별 기록
  - 참여 TOP
  - 이번 달 이벤트 목록
- 공유 문구 복사 성공 / 실패 메시지 표시
- 공유 문구 복사 버튼을 캡처용 정산 카드가 아니라 월별 정산 설정 영역에 배치
- 점령전 빠른 제목 버튼 제거
- 점령전 세부 카테고리 선택 중심으로 입력 UX 정리
- 기존 점령전 제목 데이터 호환 유지
- 길드원 활동 이력 보기를 모달 방식으로 변경
- 기존 메인 화면 하단에 활동 이력이 길게 펼쳐지던 구조 개선
- 활동 이력이 많은 길드원도 메인 화면 레이아웃이 과도하게 길어지지 않도록 개선
- 모달 내부에서 길드원 활동 요약과 활동 이력 목록 확인 가능

### 결과

- 월별 활동 정산을 길드 톡방에 공유하기 쉬워짐
- 캡처용 월별 정산 카드에 조작 버튼이 보이지 않아 화면 구성이 더 깔끔해짐
- `기타`보다 길드 운영 기록에 자연스러운 `이벤트` 표현을 사용하게 됨
- 길드원 활동 이력 확인 시 모바일 화면 사용성이 개선됨
- 활동 이력이 많은 길드원을 확인해도 전체 화면 레이아웃이 무너지지 않음
- 점령전 입력 화면의 중복 요소가 줄어듦

### 제외

- PNG 저장 기능은 구현하지 않음
- 별도 길드 연혁 페이지는 아직 추가하지 않음
- 내부 ActivityType과 schemaVersion은 변경하지 않음
- 기존 LocalStorage 데이터 구조는 유지
- 대시보드 상단에 긴 리포트 문장을 상시 표시하지 않음

## v1.4 - JSON 백업 가져오기 및 데이터 복원 안정화

### 목표

LocalStorage 기반 앱의 데이터 유실 위험을 줄이기 위해 JSON 백업 내보내기뿐 아니라 가져오기 / 검증 / 복원 기능까지 제공한다.

### 구현

- 데이터 관리 도구 구조 정리
- 새 길드원 등록 영역을 데이터 관리 도구 안으로 이동
- JSON 백업 내보내기 구조 보완
  - appName
  - appVersion
  - schemaVersion
  - exportedAt
  - members
  - activityLogs
- JSON 백업 파일 가져오기 기능 추가
- 백업 파일 구조 검증 추가
- 복원 전 백업 파일 요약 표시
- JSON 백업 파일 기반 덮어쓰기 복원 기능 추가
- 복원 전 2단계 확인 절차 추가
- 복원 완료 결과 메시지 표시
- 잘못된 파일 / 복원 취소 시 기존 데이터가 변경되지 않도록 처리
- 기존 스프레드시트 가져오기 기능 제거

### 결과

- 전체 데이터를 JSON 파일로 백업하고 다시 복원할 수 있게 됨
- 잘못된 백업 파일로 인한 데이터 손상 위험을 줄임
- 복원 전 요약과 확인 절차를 통해 실수로 기존 데이터를 덮어쓰는 위험을 줄임
- 데이터 관리 기능이 앱 내부에서 더 명확하게 정리됨

## v1.3.1 - 월별 정산 표시 보완 및 점령전 세부 카테고리 추가

### 목표

월별 활동 정산 대시보드의 표시 방식을 다듬고, 점령전 기록을 더 구체적으로 남길 수 있게 한다.

### 구현

- 대시보드 상단 카드 표시 개선
- 활동 종류별 통계 순서 개선
  - 비공정
  - 점령전
  - 기타
- 기타 0회 숨김 유지
- 이번 달 활동 기록에서 비공정 표시명 간소화
  - 오션헤븐 비공정 → 오션헤븐
  - 아우로라 비공정 → 아우로라
- 점령전 세부 카테고리 9종 추가
  - 용기
  - 신념
  - 평화
  - 신성
  - 지혜
  - 예언
  - 초심
  - 긍지
  - 역전
- 점령전 세부 카테고리 다중 선택 지원
- 활동 기록 카드와 월별 대시보드에 `점령전 (지혜, 초심)` 형태로 표시
- 기존 점령전 기록은 세부 카테고리 없이도 기존처럼 `점령전`으로 표시
- JSON 백업 앱 버전을 v1.3.1로 갱신

### 결과

- 점령전 활동을 더 세부적으로 기록할 수 있게 됨
- 월별 활동 정산에서 점령전 참여 성격을 더 잘 파악할 수 있게 됨
- 기존 기록과의 호환을 유지하면서 기록 표현력이 좋아짐

## v1.2 - 활동 종류 정리 및 길드원별 통계 추가

### 목표

실제 길드 운영에서 자주 쓰는 활동 종류 중심으로 기록 구조를 정리하고, 길드원별 활동 통계를 확인할 수 있게 한다.

### 구현

- 활동 종류를 `점령전 / 비공정 / 기타` 중심으로 정리
- 기존 `길퀘 / 이벤트` 기록은 호환용으로 유지하되 표시 / 통계에서는 기타로 취급
- 비공정 세부 종류 추가
  - 오션헤븐
  - 아우로라
- 길드원별 활동 통계 계산 로직 분리
- 길드원 카드에 활동 요약 통계 표시
- 길드원 카드 안에서 활동 이력 / 상세 통계 확인
- 활동 추가 / 수정 폼에서 참여자 검색 추가
- 활동중 / 탈퇴 길드원 분리 선택
- 탈퇴 길드원도 과거 활동 참여자로 기록 가능
- 참여자 목록에서 현재 활동 종류 기준 참여 횟수 표시
- 데이터 관리 도구 접기 / 펼치기 정리

### 결과

- 실제로 기록할 활동 종류가 단순해짐
- 길드원별 참여도를 확인할 수 있게 됨
- 탈퇴 길드원도 과거 기록에 남길 수 있게 되어 아카이브 성격이 강화됨
