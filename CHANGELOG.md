# CHANGELOG

## v1.1

### 목표

과거 기록 복원과 실사용 입력 편의성을 개선합니다.

기존에 흩어져 있던 길드원 기록을 더 쉽게 가져오고, 활동 기록 작성 중 반복 입력을 줄이며, 스크린샷 기반 기록도 함께 남길 수 있게 하는 것이 목표입니다.

### 구현 기능

- 길드원 정보 수정
- 활동 기록 스크린샷 첨부
- 디스코드 이미지 Ctrl+V 첨부
- 스프레드시트 길드원 일괄 가져오기
- 일괄 가져오기 되돌리기
- 탈퇴 길드원 일괄 복구
- 참여자 목록 빈도순 정렬
- 활동중/탈퇴 길드원 분리
- 길드원 관리 영역 접기/펼치기
- 활동 제목 빠른 입력 버튼

### 결과

v1.1에서는 과거 길드원 목록을 빠르게 복원하고, 잘못 가져온 최근 일괄 추가분을 되돌릴 수 있습니다.

탈퇴 상태로 남아 있던 길드원을 한 번에 활동중으로 복구할 수 있어, 과거 기록을 다시 이어서 쓰기 쉬워졌습니다.

활동 기록에는 스크린샷을 첨부할 수 있고, 디스코드에서 복사한 이미지를 Ctrl+V로 바로 붙여넣을 수 있습니다.

참여자 선택 목록은 기존 참여 빈도가 높은 길드원이 먼저 보이도록 정렬되어 반복 입력이 줄었습니다.

### 다음 개선 후보

- LocalStorage 데이터 내보내기/가져오기
- 활동 목록 검색
- 활동 기록 정렬 옵션
- 간단한 참여 횟수 요약
- 첨부 이미지 용량 관리 방식 개선

## v1.0

### 목표

냥춘 길드 활동을 개인적으로 기록하고 다시 확인할 수 있는 LocalStorage 기반 아카이브를 만듭니다.

길드원 관리 시스템보다는 활동 기록 보존에 집중합니다.

### 구현 기능

- 길드원 등록
- 길드원 탈퇴 처리
- 탈퇴 길드원 기록 보존
- 활동 기록 추가
- 활동 종류 선택
  - 비공정
  - 점령전
  - 길드퀘
  - 이벤트
  - 기타 메모
- 활동 참여자 체크
- 전체 활동 기록 목록 보기
- 활동 종류별 필터
- 활동 기록 수정
- 활동 기록 삭제
- 길드원별 활동 이력 보기
- LocalStorage 저장

### 결과

v1.0에서는 길드 활동을 날짜, 종류, 제목, 메모, 참여자 기준으로 남기고 조회할 수 있습니다.

탈퇴한 길드원도 삭제하지 않고 상태만 변경하므로, 과거 활동 기록의 참여자 이력이 유지됩니다.
