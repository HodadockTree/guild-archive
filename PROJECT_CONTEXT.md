# PROJECT_CONTEXT

## 프로젝트 개요

냥춘 길드 활동 아카이브는 테일즈런너 길드 `냥춘`의 활동 기록을 관리하고 보관하기 위한 모바일 중심 웹앱이다.

이 프로젝트는 단순 길드원 관리 시스템이 아니라, 길드 활동을 기록하고 월별로 정산해 나중에 다시 볼 수 있도록 하는 활동 아카이브를 목표로 한다.

핵심 기준은 다음과 같다.

- 실제 길드 운영에 도움이 되는가
- 활동 기록을 쉽게 입력할 수 있는가
- 기록된 데이터를 나중에 보기 쉽게 활용할 수 있는가
- 월별 정산과 공유에 쓸 수 있는 의미 있는 데이터인가
- 과도한 기능보다 자주 쓰는 기능을 우선하는가

## 현재 버전

현재 버전은 v1.3.1이다.

v1.3에서 월별 활동 정산 대시보드와 JSON 백업 내보내기를 추가했고, v1.3.1에서 대시보드 표시와 점령전 세부 기록을 보완했다.

## 현재 주요 기능

### 길드원 관리

- 활동중 길드원과 탈퇴 길드원을 분리해서 관리한다.
- 탈퇴 길드원도 과거 활동 참여자로 기록할 수 있다.
- 활동 기록이 없는 길드원은 삭제할 수 있다.
- 길드원 카드에서 활동 요약 통계를 확인할 수 있다.
- 길드원별 활동 이력을 확인할 수 있다.

### 활동 기록

- 활동 종류는 점령전 / 비공정 / 기타를 중심으로 한다.
- 기존 길퀘 / 이벤트 기록은 호환용으로 유지하되, 표시와 통계에서는 기타로 취급한다.
- 비공정은 오션헤븐 / 아우로라 세부 종류를 기록할 수 있다.
- 점령전은 9개 세부 카테고리를 기록할 수 있다.
  - 용기
  - 신념
  - 평화
  - 신성
  - 지혜
  - 예언
  - 초심
  - 긍지
  - 역전
- 점령전 세부 카테고리는 다중 선택을 지원한다.
- 점령전 세부 카테고리가 있는 경우 `점령전 (지혜, 초심)` 형태로 표시한다.
- 기존 점령전 기록은 세부 카테고리가 없어도 기존처럼 `점령전`으로 표시한다.
- 활동 추가 / 수정 폼에서 참여자를 검색할 수 있다.
- 참여자 초성 검색을 지원한다.
- 활동중 / 탈퇴 길드원을 분리해서 참여자로 선택할 수 있다.
- 참여자 목록에서 현재 활동 종류 기준 참여 횟수를 확인할 수 있다.

### 월별 활동 정산 대시보드

- 월을 선택해 해당 월의 활동을 정산할 수 있다.
- 월별 전체 활동 수를 표시한다.
- 참여 길드원 수를 표시한다.
- 총 참여 횟수를 표시한다.
- 비공정 / 점령전 / 기타 활동 통계를 표시한다.
- 기타는 0회일 때 숨김 처리한다.
- 오션헤븐 / 아우로라 비공정 세부 통계를 표시한다.
- 월간 참여 TOP N을 표시한다.
- 이번 달 활동 기록을 날짜 / 활동 종류 / 참여 인원 수 중심으로 요약한다.
- 최다 참여 활동에는 배지를 표시한다.
- 캡처해서 길드 톡방에 공유하기 좋은 레이아웃을 지향한다.

### 데이터 관리

- 데이터는 현재 LocalStorage 기반으로 저장한다.
- 전체 데이터를 JSON 파일로 백업 내보내기 할 수 있다.
- 백업 JSON에는 앱 이름, 버전, 스키마 버전, 백업 생성일, 길드원 목록, 활동 기록 목록이 포함된다.
- v1.3.1 기준 백업 버전은 `1.3.1`이다.

## 데이터 구조 관련 메모

### 활동 기록

활동 기록은 기본적으로 다음 정보를 포함한다.

- id
- type
- date
- title
- participantIds
- memo
- imageDataUrl
- airshipType
- conquestTypes

`airshipType`은 비공정 세부 종류를 나타내며, 오션헤븐 / 아우로라를 구분한다.

`conquestTypes`는 점령전 세부 카테고리를 나타내며 optional 필드이다. 기존 점령전 기록에는 이 값이 없을 수 있다.

기존 데이터 호환을 위해 `conquestTypes`가 없거나 비어 있는 점령전은 일반 `점령전`으로 표시한다.

- 길드원별 활동 이력 모달화
